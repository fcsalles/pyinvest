# 💰 PyInvest

Simulador de investimentos em Python que compara diferentes aplicações financeiras como CDB, LCI/LCA, poupança e FIIs.

## 🚀 Funcionalidades

* Cálculo de juros compostos
* Simulação com aportes mensais
* Comparação entre diferentes tipos de investimento
* Análise estatística para FIIs (média, mediana e desvio padrão)

## 🛠️ Tecnologias utilizadas

* Python
* Biblioteca `statistics`
* Biblioteca `random`

## 📊 Como funciona

O usuário insere:

* Capital inicial
* Aporte mensal
* Prazo (em meses)
* Taxa do CDI

O programa calcula o rendimento para cada tipo de investimento e retorna os resultados.

## ▶️ Como executar

```bash
python nome_do_arquivo.py
```

## 📈 Exemplo de uso

```text
Capital inicial: R$1000
Aporte mensal: R$200
Prazo: 12 meses
CDI: 13%
```
